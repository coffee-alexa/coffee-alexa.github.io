Vue的原生指令

  • v-html 和 v-text

    • v-html 和 很像
    • v-text 渲染时会考虑是不是有标签元素

  • v-if ,v-else-if, v-else和 v-show

    • 尽量用v-show, 性能更好,不用重复删改DOM

  • v-on

  • v-bind

  • v-for

  • v-model

    • 表单输入相关的指令

    • 对于 text, v-model的值为字符串

    • 对于 checkbox (多选), v-model的值为数组, 数组中的 某个 值 === 元素中value的值, checked就为true, 也就是勾选状态

    • 对于 radio (单选), v-model的值为字符串, 字符串 === 元素中value的值, checked就为true, 也就是勾选状态

    • 修饰符, v-model.number, v-model.trim, v-model.lazy

  • v-once

    • 对于只渲染一次的, 使用这个可以忽略虚拟DOM检测,提升性能

Vue的组件的定义

  • 注意

    • data 里面要用函数return值, 防止全局共用值
    • 子组件用props取父组件传来的值, props的值有默认属性 type, required, default
